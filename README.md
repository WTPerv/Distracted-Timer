# Distracted Timer
This creates a small overlay that floats above everything\
It containsa timer that ticks up whenever Windows is focused on any software that's not included in a whitelist set up by you 🫵\
*(There's also a mini timer that does the opposite)*\
*(For moral support)*

## How to use/install
### Option A: Executable
1) Download the latest [release](https://github.com/WTPerv/Distracted-Timer/releases)
2) Place it somewhere you won't forget
3) Execute it

### Option B: AutoHotKey Script
1) Install [AutoHotKey](https://www.autohotkey.com/)
2) Download the [script](https://github.com/WTPerv/Distracted-Timer/blob/main/distractedTimer.ahk)
3) Place it somewhere you won't forget
4) Run the .ahk file

After going through either option, you should have now a floating timer and a clock in your system tray\
Clicking the tray icon brings up a menu\
Click on "Start with Windows" if you don't wanna execute it manually every time

## Features
* Floating overlay
* Big timer counts when not focused on whitelisted software
* Small timer does the opposite
* Time is saved when you close it, but resets at midnight
* Easy to configure whitelist
* Overlay is invisible to clicks unless you enable draggable mode
* Adjustable opacity
* Adjustable scale
* Option to start with Windows
